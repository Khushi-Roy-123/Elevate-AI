# ErrorBuddy - Code Error Explainer

From Error to Clarity in Seconds

A simple web tool that explains error messages in plain English, shows corrected code, and displays visual simulations for DSA and ML concepts.

## Hackathon Details
- Hackathon: HackMatrix 2026, VIT Bhopal
- Problem ID: HM10 - Code Error Explainer
- Track: Artificial Intelligence and Machine Learning
- Team: Khushi, Jash and team

## Key Features
- Instant plain English explanation of errors
- Shows corrected code with changes highlighted
- Live visual simulations for DSA and ML
- Saves error history (works offline)
- Clean and easy to use interface

## Tech Stack
- Next.js 15 + React 19
- TypeScript
- Tailwind CSS
- shadcn/ui
- Spline for 3D background
- Canvas and SVG for animations

## How to Run
1. Clone the repository
   git clone https://github.com/YOUR-USERNAME/errorbuddy-hackmatrix.git
2. Go to the project folder
   cd errorbuddy-hackmatrix
3. Install dependencies
   npm install
4. Start the app
   npm run dev

Open http://localhost:3000 in your browser.
